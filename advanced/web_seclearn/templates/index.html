SecLearn


Learning Security has never been easier.

One of the key things about security is that you gotta keep learning, since there are always new things that you didn't know about. SecLearn is our new internal security training system. It allows you to add new security concepts to your To-Learn list and go find info about them.

Equipped with state of the art Word Embeddings algorithm Word2Vec, for every concept you learn, SecLearn recommends related concepts to you so that you can actually discover things you didn't learn about.
